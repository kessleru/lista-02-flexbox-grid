# Lista 02 - CSS Flexbox e CSS Grid

## Objetivo da atividade

Praticar a construção de layouts responsivos utilizando CSS Flexbox e CSS Grid, aplicando alinhamento, distribuição de elementos e organização bidimensional de conteúdo.

## Como cada exercício foi implementado

- **Exercício 1:** Flexbox em linha para alinhar os itens da barra de navegação horizontalmente.
- **Exercício 2:** Flexbox com quebra de linha (`flex-wrap`) para organizar cards em múltiplas linhas.
- **Exercício 3:** CSS Grid para montar um layout bidimensional de posts.
- **Exercício 4:** CSS Grid com áreas nomeadas (`grid-template-areas`) para estruturar header, main, aside e footer.
- **Exercício 5:** Combinação de Grid e Flexbox, usando Grid na estrutura principal da página e Flexbox no alinhamento interno (como menu e blocos de conteúdo).
